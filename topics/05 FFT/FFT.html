2.2. Denoising via filtering¶

In [4]:
%plot -s 560,420
T = 30;
n = 2^9;

t2 = linspace(-T/2,T/2,n+1); t = t2(1:n);

k = (2*pi/T)*[0:n/2-1 -n/2:-1]; ks = fftshift(k);

u = sech(t);
ut = fft(u);

noise = 15;
utn = ut + noise*(randn(1,n) + 1j*randn(1,n)); % in Fourier domain
un = ifft(utn);

filter = exp(-0.2*k.^2);
filtershift = fftshift(filter);

utnf = filter.*utn;
unf = ifft(utnf);

utshift = fftshift(ut);
utnshift = fftshift(utn);
utnfshift = fftshift(utnf);

subplot(2,1,1), plot(t,abs(un),'m',...
                     t,abs(unf),'g')
subplot(2,1,2), plot(ks,abs(utnshift)/max(abs(utnshift)),'m',...
                     ks,filtershift,'b',...
                     ks,abs(utnfshift)/max(abs(utnfshift)),'g');
axis([-25 25 0 1])
